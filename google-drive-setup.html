🔧 Quick Setup (5 minutes)

  1. Create Google Cloud Project:
    Go to Google Cloud Console
    Create new project → Name it "GIAC Book Indexer"
  2. Enable Drive API:
    APIs & Services → Library → Search "Google Drive API" → Enable
  3. Create OAuth Credentials:
    APIs & Services → Credentials → Create Credentials → OAuth 2.0 Client IDs
    Application type: Web application
    Authorized origins: http://localhost:3000
    Authorized redirect URIs: http://localhost:3000
  4. Update Code:
    Open src/utils.ts and replace:
    export const GOOGLE_CLIENT_ID = 'your-actual-client-id-here';
  5. Test:
    Run npm run dev → Click "Connect to Drive" → Sign in!

🎯 Features Available

  • Connect/Disconnect: OAuth-based Google account linking
  • Backup to Drive: Upload current index as JSON backup
  • Sync with Drive: Intelligent merge of local and cloud data
  • Browse Backups: View and restore from previous cloud backups
  • Auto-sync: Detects and handles conflicts automatically

🔒 Security & Privacy

  • Data stored in app-specific Google Drive folder (not visible in regular Drive)
  • OAuth scope limited to file operations only
  • No access to user's other Drive files
  • Backups encrypted in transit (HTTPS)

🚀 Production Deployment
